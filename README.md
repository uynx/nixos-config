# nixos-config

NixOS configuration for Apple Silicon (Asahi Linux) on a **MacBookPro18,3** (M1 Pro, ~1 TB SSD).

## File Structure

- `/configuration.nix`: Core NixOS configuration (boot loader, user config, hardware support).
- `/hardware-configuration.nix`: Hardware-specific configuration generated by Asahi Linux.
- `/flake.nix`: Flake configuration managing inputs (nixpkgs-weekly, home-manager, determinate systems, nixos-apple-silicon).
- `/hosts/uynx/`:
  - `home.nix`: Home Manager configuration managing user environment, tools, and Hyprland variables.
  - `brave-origin.nix`: Custom derivation packaging Brave Origin (wrapped from official Debian release, patched for Wayland and `libxcb` compatibility).

## Rebuild

To apply system changes:
```bash
reb
```

To update all flake inputs (including Determinate Nix) and rebuild:
```bash
update && reb
```
